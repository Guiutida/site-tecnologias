O que é HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar e estruturar páginas web. Não é uma linguagem de programação, mas sim uma linguagem de marcação que define a organização e apresentação do conteúdo em um navegador. Basicamente, o HTML informa ao navegador como exibir texto, imagens, vídeos e outros elementos em uma página.
